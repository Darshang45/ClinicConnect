# 🏥 ClinicConnect Architecture

## Overview

ClinicConnect is a scalable Hospital Management System (HMS) built using the MERN Stack.

The platform digitizes hospital workflows by connecting Patients, Doctors, Receptionists, Pharmacists, and Administrators through role-based dashboards.

The architecture follows a modular approach, allowing each hospital module to be developed, tested, and scaled independently.

---

# System Architecture

```
                   Browser
                      │
          React + Tailwind (Client)
                      │
          React Router + Context API
                      │
        Axios REST API Requests
                      │
        Express.js + Node.js (Server)
                      │
      Business Logic & Authentication
                      │
                MongoDB Atlas
                      │
        Cloudinary (Reports & Images)
```

---

# User Roles

ClinicConnect currently supports five different user roles.

1. Patient
2. Doctor
3. Receptionist
4. Pharmacist
5. Administrator

Each role has its own dashboard and permissions.
